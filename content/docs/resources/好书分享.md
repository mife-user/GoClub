# 好书分享

这里整理了一些适合 Go 开发、计算机基础和工程进阶阶段反复阅读的书。

如果你正在准备面试、补基础，或者希望把零散知识串成体系，可以先从下面几本开始。

## Go 语言

### Go 语言圣经

适合刚系统接触 Go 语言时阅读。内容覆盖语言基础、数据结构、并发、测试和工程实践，适合作为入门到进阶阶段的主线参考书。

- 阅读方向：语法基础、常用标准库、并发模型
- 适合阶段：Go 入门到中级
- 在线阅读：https://gopl-zh.github.io/index.html

<img src="/pictures/gopl-cover.png" width="260" alt="Go语言圣经封面" />

### Go 程序员面试笔试宝典

这份资料更偏面试导向，适合在掌握基础语法后查漏补缺。它对高频知识点覆盖比较集中，比较适合面试前冲刺和专题复习。

- 阅读方向：高频八股、面试题、原理串讲
- 适合阶段：求职准备、阶段性复盘
- 在线阅读：https://golang.design/go-questions/

<img src="/pictures/go-questions-cover.jpg" width="260" alt="Go程序员面试笔试宝典封面" />

### Go 语言原本

如果你已经不满足于“会用”，想进一步理解运行时、调度器、内存分配和垃圾回收，这本资料很值得反复翻。

- 阅读方向：GMP 调度、内存管理、垃圾回收、底层实现
- 适合阶段：Go 进阶、面试拔高、工程深入
- 在线阅读：https://golang.design/under-the-hood/

<img src="/pictures/go-under-the-hood-cover.png" width="260" alt="Go语言原本封面" />

### Go 语言设计与实现

这本资料更适合在已经掌握基础语法之后阅读。它会从编译原理、语言特性到运行时机制逐步展开，帮助你把“会写 Go”进一步提升到“理解 Go 为什么这么设计”。

- 阅读方向：编译过程、语言特性、运行时实现、内存与调度
- 适合阶段：Go 进阶、源码阅读、底层原理学习
- 在线阅读：https://draven.co/golang/

<img src="/pictures/Go语言设计与实现.png" width="260" alt="Go语言设计与实现封面" />

## 阅读建议

- 刚开始学 Go 时，先把《Go 语言圣经》作为主线阅读材料。
- 准备面试时，把《Go 程序员面试笔试宝典》当作专题复习清单。
- 想理解底层机制时，再重点看《Go 语言原本》中的运行时和内存相关章节。
- 如果你想把语法、编译过程和运行时实现串起来，可以继续看《Go 语言设计与实现》。
