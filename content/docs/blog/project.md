# GoClub项目介绍

GoClub 是一个围绕技术面试准备与系统复习搭建的内容站点，主要聚焦面试真题、八股总结、资料视频、项目推荐，以及视频配套文章等内容。

这个项目使用 Hugo 生成静态页面，通过 GitHub Pages 对外发布，当前主题为 `hugo-book`。整体定位不是传统博客，而是更偏向资料归档、知识整理和面试复习的文档型站点。

## 这个项目想做什么

GoClub 主要想把常见的面试准备内容系统化整理出来，减少重复搜集资料和到处翻找笔记的时间。

站点会重点覆盖以下几个方向：

- 面试真题
- 八股知识总结
- 资料与视频推荐
- 项目推荐
- 视频配套文章

这些内容会尽量按照主题、难度和使用场景去组织，方便在准备面试、补基础和做项目时快速查阅。

随着内容逐步积累，GoClub 会从一篇项目介绍开始，慢慢扩展成一个更完整的面试与学习资料站点。

## 当前技术方案

- 静态站点生成器：Hugo
- 部署平台：GitHub Pages
- 页面主题：hugo-book
- 内容格式：Markdown

这套方案的优点是简单、轻量、维护成本低，很适合长期沉淀结构化内容。

## 接下来会更新什么

接下来我会优先补充以下内容：

1. 高频面试真题与答案整理
2. 常见八股知识专题总结
3. 值得反复观看的资料和视频推荐
4. 适合练手和面试讲解的项目推荐
5. 视频内容对应的补充文章与延伸说明

## 为什么从这个页面开始

目前站点还处在初始化阶段，所以先保留一篇项目介绍页，先把整体方向、内容边界和结构规划说明清楚，再逐步补充更具体的文章和栏目。

如果你现在正在看到这个页面，说明 GoClub 已经成功完成了 Hugo 与 GitHub Pages 的基础搭建。
