# 网站分享

在这篇笔记中，我整理了几个在学习与日常开发过程中极其有用的优质网站。希望能对大家的学习和工作有所帮助。

## Git 工具

这是一个**沙盒互动式**的 Git 学习网站。它以游戏通关的可视化方式，直观地展示每一次提交背后的分支与节点转移。

- [链接](https://learngitbranching.js.org/?demo=&locale=zh_CN)

## 开源图书

以下三本是 Go 语言领域极具影响力的开源图书和笔记，它们由浅入深，覆盖了基础入门、高频面试、底层原理：

1. **Go 语言圣经**  
   《The Go Programming Language》的开源中文翻译版，内容系统全面。  
   - [**链接**](https://gopl-zh.github.io/index.html)  
   <br>
   <img src="/pictures/go-questions-cover.jpg" width="280" alt="Go语言圣经" />
2. **Go 程序员面试笔试宝典**  
   高质量 Go 面试题库，深入解析核心考点，是求职备考与日常查漏补缺的极佳指南。  
   - [**链接**](https://golang.design/go-questions/)  
   <br>
   <img src="/pictures/go-under-the-hood-cover.png" width="280" alt="Go程序员面试笔试宝典" />

3. **Go 语言原本**  
   专注剖析底层实现，带你看懂 GMP 调度、内存分配和垃圾回收等底层架构。  
   - [**链接**](https://golang.design/under-the-hood/)
