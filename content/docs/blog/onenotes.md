# Go语言相关笔记

这里整理了 Go 语言相关语法知识，除了基础内容，也包含一些常见框架与工程实践主题，例如 `os`、`err`、`fmt`、`bufio`、`time`、并发、多文件操作，以及 `gin`、`zap`、`go-zero`、`mysql`、`gorm`、`redis`、`docker`、`rabbitmq` 等。

这份笔记会随着学习过程持续更新。如果你发现错误，欢迎指出或参与补充。

## 使用方法

由于原始笔记长期在深色模式下维护，直接通过网页查看时体验可能一般。建议点击对应链接后下载并在本地使用 OneNote 查看。
