# GoClub

GoClub 是一个围绕 Go 语言学习、项目实践和技术整理搭建的个人站点。

这个项目使用 Hugo 生成静态页面，通过 GitHub Pages 对外发布，当前主题为 `hugo-book`。整体定位不是传统的时间流博客，而是更适合做文档、笔记和项目记录的知识库型网站。

## 这个项目想做什么

GoClub 主要承载三类内容：

- Go 语言基础与进阶学习笔记
- 个人项目的开发记录与复盘
- 工具使用、部署经验和踩坑总结

随着内容逐步积累，这个站点会从一篇项目介绍开始，慢慢扩展成一个更完整的个人技术站点。

## 当前技术方案

- 静态站点生成器：Hugo
- 部署平台：GitHub Pages
- 页面主题：hugo-book
- 内容格式：Markdown

这套方案的优点是简单、轻量、维护成本低，也很适合把学习内容持续沉淀下来。

## 接下来会更新什么

接下来我会优先补充以下内容：

1. Go 学习路线和基础语法整理
2. 常用项目结构与工程化实践
3. Git、部署和开发环境相关笔记
4. 个人项目开发过程中的问题记录

## 为什么从这个页面开始

目前站点还处在初始化阶段，所以先保留一篇项目介绍页，确保网站结构清晰、访问正常，再逐步补充更多文章和栏目。

如果你现在正在看到这个页面，说明 GoClub 已经成功完成了 Hugo 与 GitHub Pages 的基础搭建。
